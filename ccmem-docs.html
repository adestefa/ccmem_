Installation

Prerequisites

  • Node.js 16+ and npm
  • Claude Code CLI
  • Git (for version control)

Quick Setup

# Clone the repository
git clone https://github.com/adestefa/ccmem_.git
cd ccmem

# Install dependencies
npm install

# Start the CCMem MCP server
npm start

MCP Configuration

Add CCMem to your Claude Code MCP servers by creating or updating .mcp.json:

{
    "mcpServers": {
        "ccmem": {
            "command": "npx",
            "args": ["tsx", "/path/to/ccmem/main.ts"],
            "description": "CCMem - Prime Swarm Orchestration System"
        }
    }
}

Core Philosophy

"We recommend against any action that would be illogical and break the application. We always prioritize field summary over code changes. Prime functions with pure logic, serving as guardian of application integrity."

Operational Modes

Builder Mode

Green field projects - optimized for rapid development and innovation with logical safeguards.

Maintainer Mode

Brown field projects - prioritizes application stability and integrity with minimal risk tolerance.

Command Reference

Prime Swarm Orchestration

/ccmem-prime-swarm NEW

Deploy specialized sub-agents with Prime oversight

/ccmem-prime-swarm "implement user authentication with OAuth2"
/ccmem-agent-monitor NEW

Monitor and control sub-agent activities

/ccmem-agent-monitor status
/ccmem-agent-monitor halt

Usage Examples

Basic Workflow

# 1. Initialize project
/ccmem-init

# 2. Set operational mode
/ccmem-set-mode maintainer

# 3. Deploy swarm for complex task
/ccmem-prime-swarm "implement secure user registration with email verification"

# 4. Monitor progress
/ccmem-agent-monitor status

# 5. Analyze before deployment
/ccmem-logical-analysis "deploy user registration to production"

Safety-Critical Operations

# High-risk database migration
/ccmem-logical-analysis "migrate user table to new schema with zero downtime"
# Expected: REJECT or CAUTION with feature branch requirement

# Validate sub-agent action
/ccmem-agent-monitor validate_action "alter table users add column email_verified boolean" db_agent

# Emergency stop if needed
/ccmem-agent-monitor halt
